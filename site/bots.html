How it works

A breakout entry, then a disciplined recovery basket.

Both bots run the same engine: a pending one-cancels-other (OCO) breakout. A BUY STOP is set above price and a SELL STOP below — whichever the market reaches first fills, and the other is cancelled. The filled side then runs a Fibonacci recovery basket that closes at a combined take-profit, with an emergency drawdown guard watching the whole time.

1

It sets the trap

When flat, the bot places a BUY STOP above and SELL STOP below the current price — set, not entered.

2

One side fills

Price breaks one way; that order fills and rides the move. The opposite pending is cancelled automatically.

3

The basket manages

If price moves against the fill, the bot adds measured recovery levels, aiming for a combined take-profit.

4

The guard watches

An emergency drawdown guard can cut the basket if floating loss exceeds your set % of balance.

You keep full custody — the bot runs on your own MetaTrader 5 / broker account. It executes; you stay in control. Running it is how you learn ranges, volatility, entries, exits, and above all risk management and discipline.

Getting started

Set up in minutes — then test on demo.

1

Get MetaTrader 5

Install MT5 from your broker (Deriv recommended for low XAUUSD spreads).

2

Install the bot

Drop the .mq5 file into MQL5 → Experts, compile (F7), refresh the Navigator.

3

Attach & allow

Open the right chart, drag the EA on, tick Allow Algo Trading. You'll see the breakout lines set.

4

Demo first

Run it on a demo account and in the Strategy Tester before risking real funds. Always.
